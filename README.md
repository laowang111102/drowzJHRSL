# 基于Python的Django-vue高校教室管理系统 python693

## 项目概述

本项目是基于Python的Django-vue技术栈开发的高校教室管理系统，旨在为高校提供一套完整的教室资源管理解决方案。

## 技术栈

- 编程语言：Python
- 开发工具：PyCharm
- 后端框架：Django
- 前端框架：Vue
- 数据库版本：MySQL 5.7+
- 数据库工具：Navicat、SQLyog

## 功能模块

### 系统角色

#### 管理员
- 首页
- 个人中心
- 教师管理
- 学生管理
- 教室信息管理
- 教师申请管理
- 学生申请管理
- 课时表管理

#### 学生
- 首页
- 个人中心
- 教室信息管理
- 学生申请管理
- 课时表管理
- 学生取消预约管理

#### 教师
- 首页
- 个人中心
- 教室信息管理
- 教师申请管理
- 课时表管理
- 教师取消预约管理

## 运行环境

- Python环境
- Django框架
- Vue框架
- MySQL数据库

## 部署步骤

1. 环境准备：确保Python、Django、Vue、MySQL等环境正确安装和配置。
2. 数据库搭建：使用Navicat或SQLyog创建数据库，并导入初始数据。
3. 后端部署：根据配置文件和数据库信息部署后端服务。
4. 前端部署：将Vue项目构建后部署到Web服务器。
